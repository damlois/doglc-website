## DOGLC website (static)

### Structure

- `index.html`: page content/markup
- `assets/css/main.css`: site styles
- `assets/js/main.js`: behavior (tabs, nav, form submit)
- `assets/img/`: images used by the page

### Run locally

Any static server works. For example:

```bash
npx serve .
```

Or with Python:

```bash
python -m http.server 8080
```
